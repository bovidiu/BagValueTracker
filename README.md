![BagValueTracker WoW addon](https://github.com/bovidiu/BagValueTracker/blob/main/icon.png)

BagValueTracker shows the sell value of items in each of your bags as an overlay when you open them, and displays available slot counts on bag icons when they are closed.

## Install

- [CurseForge](https://secure.lockn.link/curseforge-bag-value-tracker)
- [Wago](https://secure.lockn.link/wago-bag-value-tracker)

## Features by version

### The War Within / Retail
- Per-bag vendor value displayed when a bag is opened (bags 1–4 enabled by default; backpack excluded)
- Combined bags mode supported — when all bags are shown in a single frame, a single total value is displayed
- Available slot count overlaid on each bag icon when bags are closed

### Cataclysm Classic
- Per-bag vendor value displayed when a bag is opened (all bags enabled by default)
- Available slot count overlaid on each bag icon when bags are closed
- Binds the **B** key to open all bags on first login

### Mists of Pandaria Classic
- Per-bag vendor value displayed when a bag is opened (all bags enabled by default)
- Available slot count overlaid on each bag icon when bags are closed
- Binds the **B** key to open all bags on first login

## Optional dependency

Install [Auctionator](https://www.curseforge.com/wow/addons/auctionator) to show auction house prices instead of vendor sell prices. Falls back to vendor value automatically if Auctionator is not installed or has no price data for an item.
